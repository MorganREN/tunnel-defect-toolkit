# Artifact Schema

This document describes the main V1 output artifacts.

## `manifest.csv`

Generated by:

```bash
tdt manifest CONFIG --out manifest.csv
```

Columns:

- `schema_version`: currently `tdt-manifest-v1`
- `image_id`
- `image_path`
- `mask_path`
- `split`

## `splits.csv`

Generated by:

```bash
tdt split CONFIG --out splits/
```

Columns:

- `schema_version`: currently `tdt-splits-v1`
- `image_id`
- `image_path`
- `mask_path`
- `split`

## `tile_manifest.csv`

Generated by:

```bash
tdt tile CONFIG --out tiles/
```

Columns:

- `schema_version`: currently `tdt-tile-manifest-v1`
- `tile_id`
- `source_id`
- `split`
- `image_path`
- `mask_path`
- `x0`, `y0`, `x1`, `y1`
- `tile_width`, `tile_height`
- `source_width`, `source_height`
- `tile_size`
- `stride`

## `morphology_descriptors.csv`

Generated when `tdt analyze --with-morphology` is used.

Each row is one connected component in a semantic mask.

Columns:

- `class_id`
- `instance_id`
- `area`
- `perimeter`
- `compactness`
- `elongation`
- `eccentricity`
- `solidity`
- `bbox_width`
- `bbox_height`
- `skeleton_length`
- `mean_width`
- `orientation`
- `image_id`
- `class_name`
