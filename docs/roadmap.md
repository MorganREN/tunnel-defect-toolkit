# Roadmap

## `0.1.0`: Quiet PyPI Release

- Stable semantic-mask conversion, validation, morphology profiling, reporting,
  overlay, splitting, and paired tiling workflows.
- Licensed repository-only example, documented output schemas, and installable CLI.
- Release intended for early users and reproducibility checks.

## `0.2.0`: Scientific Definition And Adapter Coverage

- Optional image-scale calibration and physically meaningful unit output where
  calibration metadata is provided.
- Expanded annotation-quality diagnostics and descriptor sensitivity summaries.
- Additional common semantic-mask directory and structured-annotation adapters.

## `0.3.0`: Public Promotion Threshold

- Verified workflows across multiple publicly reusable or expressly licensed
  datasets.
- Documented performance and descriptor stability results.
- Complete tutorials, methods documentation, and end-to-end case studies.

This is the earliest intended point for broad public promotion.

## `1.0.0`: Stable Research Infrastructure

- Frozen public API and artifact-schema compatibility policy.
- Long-term archived release, formal citation record, and maintained release
  automation.
- Sustained cross-dataset reproducibility verification.
