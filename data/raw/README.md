# Raw Dataset Placement

Place real datasets here manually. This directory is intentionally separate from
the toy dataset and should not be used to commit restricted or private data.

Suggested layout:

```text
data/raw/
  tongji/
    images/
    annotations/
    masks/
    splits/
  codebrim/
    images/
    annotations/
    masks/
  rdd2022/
    images/
    annotations/
    masks/
```
